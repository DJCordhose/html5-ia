For the Canvas 2D chapter you'll find 6 examples, each with a corresponding folder.
To run one of the folders simply open it up and run the index.html file in Firefox or Google Chrome on a desktop.

Contents:
- Simple Shapes: Create a basic smiley face with squares.
- Paths: Create a smiley face with circular elements via paths.
- Images: Load an object into Canvas and display it.
- Text: Basic demo of writing with text in Canvas.
- Animation: Bare bones animation with intervals.
- Breakout: Complete game with all necessary files.
